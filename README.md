# ssm
SpringMVC+Spring+MyBatis及其他常见开源jar

**基础功能**
 ![image](https://github.com/yongzhian/ssm/blob/master/src/docs/demo.png)

**基础功能**
1. 基于Jquery的系统登录
1. 基于MySql的DB查询
1. 基于MongoDB的Excel文件导出和流导出（POI）
1. log4J2的日志
1. 基于Jetty的Web测试
1. 基于MockMvc的Controller单元测试
1. 基于MyBatisGenerator进行自动化生成DAO
1. 配置优化
1. 基于PageHelper分页查询
1. 添加CRUD基本功能
1. 添加BootStrap框架
